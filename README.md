# 🚗 Car Price Prediction Web App

This project is a Machine Learning-based web application that predicts the price of a used car based on user inputs such as year, present price, kilometers driven, fuel type, seller type, and transmission type.

## 📌 Project Overview
The goal of this project is to build an end-to-end ML model and deploy it as a web app using Streamlit.

## 🧠 Machine Learning Model
- Algorithm: Random Forest Regressor / Regression Model
- Trained on car dataset
- Predicts estimated resale value of used cars

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle
